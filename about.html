About Us

Mykal's MC Server is a new 1.19.2 Survival moded server with the famous create mod. We are currently looking for helpers and admins to help make this a better place for us all.

Where is the Mykals MC Minecraft Server being hosted?

The Mykals MC server is currently hosted in United States of America with an uptime of 97%.

What GameModes can you play on mykals?

What gamemodes can I play on the Mykals MC Minecraft Server? You can play Moded survival with the create mod which adds a variety of tools and blocks for Building, Decoration and Aesthetic Automation.

What Minecraft game version does Mykals MC server support?

Mykals MC supports Minecraft version: 1.19.2. it also supports bedrock just use this tutorial
